Board layout & terminals

Overview

Before wiring up the Morpheus Drive, get familiar with where each terminal lives on the board. The diagram below labels every connection point referenced in the next video.

Legend

M1 +/−
Motor 1 terminals (left side)
M2 +/−
Motor 2 terminals (right side)
V+ / V−
Battery power input
RST
Reset button. Press once power is wired in.
STAT
Status LED. Solid blue means the system is ready.
